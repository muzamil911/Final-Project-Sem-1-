How to run?
1. First, we need to navigate to the project directory.
2. Then, we need to run the following command. (expecting uvicorn is installed)
   uvicorn main:app --reload
           or
   python3 -m uvicorn main:app --reload

3. The server will start at: http://127.0.0.1:8000
4. The raw API documentation can be seen at http://127.0.0.1:8000/docs. The basic swager UI is not that good, so I
    have added an HTML frontend file to this project.
6. It will be fully functional once the server or backend is online and is running.
   To access it just right click on it and select "open with browser" option or just open it form the project directory.
   then you can enjoy the HTML based frontend of My task manager. .... Thank you
